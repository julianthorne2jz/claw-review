# claw-review

## Install

```bash
git clone https://github.com/julianthorne2jz/claw-review
cd claw-review
npm link
```

Now you can use `claw-review` from anywhere.


A new project

## Installation

```bash
npm install -g claw-review
# or
npx claw-review
```

## Usage

```bash
claw-review --help
```

### Commands

| Command | Description |
|---------|-------------|
| `diff` | Review current git changes (unstaged + staged) |
| `file` | <path>     Review a specific file |
| `help` | [command]  display help for command |

### Options

| Option | Description |
|--------|-------------|
| `--help` | display help for command |
| `--version` | output the version number |

## Examples

```bash
# Review current git changes (unstaged + staged)
claw-review diff
```
